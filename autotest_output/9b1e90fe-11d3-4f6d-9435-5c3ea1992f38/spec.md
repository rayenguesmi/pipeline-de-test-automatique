📄 Projet : Cloudflare – Plateforme Web & Services Cloud

URL : https://www.cloudflare.com/fr-fr/

🌐 Feature F-001 : Affichage de la page d’accueil

L'utilisateur peut consulter la page d’accueil du site.

Critères d'acceptation :
La page affiche :
Header (menu navigation)
Sections de présentation des services
Boutons CTA (ex : “Commencer”, “Contact”)
Présence de contenu textuel et visuel
Le site est responsive
Flux positif :
Accéder à l’URL principale
Visualiser les sections du site
Flux négatifs :
URL incorrecte → page erreur
Problème de chargement → page partiellement affichée
🧭 Feature F-002 : Navigation dans le menu principal

L'utilisateur peut naviguer entre les différentes sections du site.

Critères d'acceptation :
Menu contient :
Produits
Solutions
Ressources
Entreprise
Tarification
Les liens sont cliquables
Redirection correcte vers les pages correspondantes
Flux positif :
Cliquer sur un menu
Accéder à la page associée
Flux négatifs :
Lien cassé → page 404
Mauvaise redirection
☁️ Feature F-003 : Consultation des produits Cloudflare

L'utilisateur peut consulter les services proposés.

Critères d'acceptation :
Liste de services affichée (ex : CDN, sécurité, DNS)
Chaque service contient :
Nom
Description
Lien “En savoir plus”
Flux positif :
Accéder à “Produits”
Cliquer sur un service
Voir les détails
Flux négatifs :
Service non cliquable
Mauvaise redirection vers détail
